# 力量体系：盘龙（开篇 23 章节选）

玉兰大陆是尚武世界，男人的地位由实力与金钱决定。力量体系分为两条主线 + 一个共同巅峰：

- **战士**：分九个级别。靠修炼斗气密典或纯粹锻炼肉体提升。六级战士可单手挥三四百斤石锁、一拳碎青石；九级战士在芬莱王国即为第一强者。普通平民修不起珍贵的斗气密典，只能靠从小锻炼身体、打熬力气这条「纯肉体路线」，理论上纯肉体也可达六级乃至更高（林雷曾爷爷曾纯靠身体修至七级战士）。斗气分属性（火/水/风/地/雷/光明/黑暗等），储于肚脐下方一拳处的核心三角区。
- **魔法师**：分九个级别——一二级初级、三四级中级、五六级高级，七八九级极为恐怖。魔法师资质稀缺，约一万人中才有一个。施法依靠体内「魔法力」（精纯炼化的天地元素）与「精神力」（灵魂之力，控制能力），并调动外界天地元素——禁忌魔法中天地元素能量占比高达 99%。精神力足够强可瞬发、无需魔法咒语。同级魔法师实力略胜战士，地位高一级（因破坏力极大）。
- **圣域**：战士九级、魔法师九级之上的共同巅峰，即传说中的圣域强者 / 圣魔导师。拥有「毁天灭地」级别的力量——可融化冰山、崩溃高山、毁灭百万人口城池、降下陨石。圣域强者寿命近乎无限（普通人寿 120-130 岁，强者 200-400 岁，人类极限 500 岁，唯圣域可近无限）。圣域战士可凌空而立，圣域魔法师可施展灭城的禁忌魔法。

**四大终极战士**：传说数千年前出现过四大终极战士——龙血战士、紫焰战士、虎纹战士、不死战士，实力可媲美巨龙、属圣域级别。四大终极战士其实代表四个古老的血脉家族。

**魔兽等级**：魔兽亦分级别。本节选出现：四级魔兽狮鹫、五级魔兽嗜血铁牛、七级魔兽迅猛龙、九级乃至圣域级别的黑龙、魔兽影鼠（出场为幼崽）。收服魔兽需两个条件——正面击败令其心甘情愿臣服 + 布置灵魂契约魔法阵（需七级魔法师，或花一万金币购买有价无市的灵魂契约卷轴）。
