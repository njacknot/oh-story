# 地理：盘龙（开篇 23 章节选）

- **玉兰大陆**：故事所在大陆。有三大险地——第一山脉「魔兽山脉」（贯穿大陆南北、过万里之长、内有圣域魔兽）、第二山脉「落日山脉」、第一森林「黑暗之森」。
- **魔兽山脉以西**：12 王国、32 公国，主要分两派——以芬莱王国为首、属光明教廷的「神圣同盟」（六大王国 + 诸多公国），与以黑狮王国为首、属黑暗教廷的「黑暗同盟」，两派战伐不断。
- **魔兽山脉以东**：四大帝国（中央玉兰帝国、东南莱茵帝国、东边罗奥帝国、北方奥布莱恩帝国）、六大王国及诸多公国，皇权至上，不受光明/黑暗教廷影响。
- **乌山镇**：故事主场景。芬莱王国境内、魔兽山脉西方的一个普通小镇，由衰败的巴鲁克家族领有，约五千余人口。镇东北有高千米的乌山。
- **普昂帝国**：八千多年前立国、维持近三千年、五千多年前已灭国的古帝国，当年领土相当于今神圣同盟 + 黑暗同盟全境。德林柯沃特即普昂帝国的圣域魔导师。
